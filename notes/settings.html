Settings

Global settings are located in the pkrc file at the root of your vault. Settings are just frontmatter variables.

Info
  • Each setting can be overrided on a page basis.
  • When you edit a setting, you have to re export your file.

general

pk_folder: /Users/louis/sites/PublishKit
site_id: insert_your_id
site_name: PublishKit
site_assets: /assets
description: Welcome to PublishKit

layout

fluid: false
header: true
footer: true
toc: true
rtoc: true

search

search: true
search_chars: 3
search_fuzzy: 0.2
search_padding: 20
search_max_results: 5

styles

css

You can override css variables.

--font-family: Cousine, sans-serif
--primary: '#0ed519'
--primary-hover: '#0bb714'
Info

You can choose any fonts from cdnfonts.com
Go to the font page that you like, for exemple this one.
font-family: 'Ambery Garden', sans-serif;

Make sure you remove the quotes ' and the ; 

--font-family: Ambery Garden, sans-serif

codeblocks

You get syntax highlighting out of the box with highlight.js
You can choose from the available themes here.
